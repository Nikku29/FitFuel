## About This Project
FitFuel AI Guide is a completely original project developed from scratch using modern web technologies. It represents a comprehensive fitness companion application with AI-powered personalization features.

## Technologies Used
- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + shadcn/ui components
- **State Management**: React Query + Context API
- **Authentication**: Firebase Auth
- **Database**: Firestore
- **AI Integration**: DeepSeek/OpenAI API
- **Animations**: Framer Motion + Lottie React
